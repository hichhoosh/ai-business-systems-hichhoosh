# AI Business Systems — Hichhoosh

> A curated library of production-ready AI agents for business strategy, market research, and high-converting copywriting. Built for solo SaaS founders, digital entrepreneurs, and agency owners.

---

## What is this repo?

This repository contains a growing collection of **AI system prompts (agents)** designed to replace expensive consultants and copywriters for:
- Validating SaaS niches and product ideas
- Writing high-converting landing pages and sales copy
- Designing offers, pricing, and positioning
- Mapping marketing funnels and automation workflows

Each agent is plug-and-play: copy the prompt into Claude, ChatGPT, or Google AI Studio as a **System Prompt**, then start chatting.

## Agents Available

### 1. Market Research Copilot
**File:** `market-research/agent-niche-analyzer-hichhoosh-v1.txt`

Delivers a complete, actionable market analysis for any niche or product idea.

**What it produces:**
- Niche Overview (market size, trends, growth signals)
- Target Audience (5 pain points + 5 desires)
- Competitor Landscape (table format with pricing & gaps)
- Opportunity Score table (/10)
- Recommended Entry Angle + Next Step (action within 48h)

**Best used with:** Claude 3.5+, Perplexity, Google AI Studio

---

### 2. Landing Page & Copywriting Architect
**File:** `copywriting/agent-landing-page-hichhoosh-v1.txt`

Produces a full landing page structure + copy for any SaaS product, digital offer, or service.

**What it produces:**
- 8-section page structure (Hero → Problem → Solution → How It Works → Social Proof → Pricing → FAQ → CTA)
- 2 headline variants per section (A/B testing ready)
- Psychological trigger note per section
- SEO Meta Block (title tag, meta description, keywords)
- Design tokens for developer handoff (layout, colors, fonts)

**Best used with:** Claude 3.5+, ChatGPT-4o

## How to Use

1. Open your preferred AI tool (Claude, ChatGPT, Google AI Studio)
2. Go to **System Prompt** or **System Instructions**
3. Copy the full content of the `.txt` file you want to use
4. Paste it as the system prompt
5. Start a new conversation and describe your project

The agent will ask you clarifying questions before producing any output.
